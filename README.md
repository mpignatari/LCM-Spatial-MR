# LCM-Spatial-MR (Oregon Marine Reserves) — factor analysis + spatial models + LCM covariates

This repository contains analysis code and supporting files for my paper work on
preference segmentation for Oregon Marine Reserves using:
- confirmatory / exploratory factor analysis (attitudinal constructs),
- spatial diagnostics and spatial GAM surfaces,
- latent class model (LCM) covariates / class-membership specifications (Latent GOLD / related inputs).

**Data are not included in this GitHub repository** (see **Data access** below).
